## Project Summary
This project is a beautifully crafted, responsive digital portfolio piece designed to make a strong visual impact. Utilizing advanced vanilla CSS techniques like glassmorphism and dynamic background gradients, it serves as a lightweight, performant, and stunning online presence. The page includes real-time JavaScript functionality and requires no external frameworks, ensuring incredibly fast load times and straightforward customization.

## Features
- **Dynamic Clock:** Real-time clock updating every second.
- **Current Date:** Full date display automatically matching the user's local date.
- **Glassmorphism Design:** Beautiful frosted-glass card effect with backdrop blur.
- **Animated Gradient:** Deep purple and indigo tones with a seamless, sweeping 15-second loop.
- **Floating Orbs:** Three colored, heavily blurred floating orbs providing visual depth.
- **Interactive Animations:** Smooth staggered entrance animations and a responsive 3D-lift hover effect on the card.
- **Typography:** Built using Google Fonts (Outfit & Inter) for a sleek, modern aesthetic.
- **Responsive Layout:** Fluid layout that adapts perfectly to both mobile and desktop screens.

## File Structure
- `index.html`: The main entry point for the webpage. It contains the complete HTML structure, CSS styling, and JavaScript time-tracking logic neatly packaged into one single file.

## Getting Started
To view this project, simply double-click the `index.html` file to open it in any modern web browser. No local server, build tools, or dependencies are required.

## Customization & Theming
You can easily customize properties within `index.html`. 
The core CSS design system utilizes custom variables in the `:root` pseudo-class for quick theme adjustments, and the primary accent gradient used on the name can be adjusted in the `h1` CSS block.
